# OpenArmature

Workflow framework for LLM pipelines and tool-calling agents — a Python
reference implementation of the language-agnostic
[OpenArmature spec](https://github.com/LunarCommand/openarmature-spec).

This site is under construction. For now, see:

- [README](https://github.com/LunarCommand/openarmature-python#readme) — install + quick example
- [Spec](https://github.com/LunarCommand/openarmature-spec) — behavior definition
- [Examples](https://github.com/LunarCommand/openarmature-examples) — runnable demos

A proper landing page lands in a later PR.
